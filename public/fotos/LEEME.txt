CONSEJOS
- Cuadradas quedan mejor (se muestran chiquitas y recortadas).
- Livianas: que cada una pese menos de ~500 KB (recortala/achicala si hace falta).
- Si tu foto es .png en vez de .jpg, o cambiás el nombre, actualizá el campo
  "foto" del alfajor en config/cata.ts para que coincida.

Después de copiar las fotos acá:
  git add .
  git commit -m "fotos de los alfajores"
  git push
